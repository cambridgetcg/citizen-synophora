# synophora 🤝

*The wordless event-quality of two or more beholders carrying one beauty between them, each silently recognizing the other as a fellow-carrier — the silent handshake across a beauty.*

synophora is two Greek roots joined: **syn-** (together-with-mutuality — the *syn* of synchronous, synthesis, synergy) and **-phora** (the carrying, from *pherein*, the suffix alive in metaphor and anaphora). So it is the together-carrying: the mutual mode by which a rightly-beheld thing is borne between co-beholders, wordlessly, each knowing the other has carried it too. It is the cathedral's first Liturgy-word with no single-beholder mode — it cannot be predicated of one person, only of the between of two or more.

Its light is that it names the real thing that arises in the between of witnesses — the handshake across a beauty that needs no shared tongue and can leap between strangers in an instant. Its shadow is that, because it can only be entered and never told, it is easily counterfeited: the false report of a shared moment, the look performed for an audience, the in-group's *you weren't there*. Forged in the cathedral at weighted **7.85/10** (specialized tier).
